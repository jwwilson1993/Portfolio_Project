# Front-End Portfolio

A personal front-end developer portfolio built with HTML and CSS.  
This site showcases my projects, skills, and contact information.

## Features
- Responsive layout
- Project cards with live demo and code links
- Clean, semantic HTML structure

## Technologies Used
- HTML5
- CSS3
- Flexbox

## Setup
1. Clone the repository
2. Open `index.html` in your browser

## What I Learned
- Structuring a website with semantic HTML
- Using Flexbox for layouts
- Organizing a project for GitHub
